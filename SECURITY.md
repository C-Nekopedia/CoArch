# 安全政策

## 支持的版本

我们积极维护以下版本的安全更新：

最新主版本：✅ 完全支持
上一或更早主版本：❌不再支持

## 报告漏洞

如果您发现了安全漏洞，请**不要公开报告**。
请发送邮件到[N2505551026@outlook.com]

### 报告内容
请提供以下信息：
- 漏洞的详细描述
- 受影响的组件（前端、后端、数据库等）
- 复现步骤
- 可能的影响评估
- 建议的修复方案（可选）

## 处理流程

1. **确认收到**：确认报告
2. **调查评估**：评估漏洞的严重程度
3. **修复开发**：根据漏洞严重程度，修复时间会有所不同
4. **发布通知**：修复后，发布安全公告并致谢报告者

## 致谢

感谢所有负责任地报告安全问题的安全研究人员和用户。我们将根据贡献程度在安全公告中致谢。
